# Social Test Mern

## Prueba de Red Social Utilizando el Stack MERN

## Tecnologias

* Bootstrap 4
* React
* Timeago.js
* React-Router

## Deploy

```bash
cd backend
yarn start | npm start
cd ../frontend
yarn start | npm start
```
